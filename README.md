# 各类发币方法实践

基于OpenZeppelin 5.0版本合约库，部署各类代币。

ERC20代币类型：

- ERC20可铸造代币
- ERC20可销毁代币
- ERC20可暂停代币
- ERC20可授权代币
- ERC20可投票代币
- ERC20闪电贷代币

ERC721代币类型：
- ERC721可铸造代币
- ERC721可销毁代币
- ERC721可暂停代币
- ERC721可投票代币
- ERC721可枚举代币
- ERC721可存储代币

ERC1155代币类型：
- ERC1155可铸造代币
- ERC1155可销毁代币
- ERC1155可暂停代币
- ERC1155可跟踪代币
- ERC1155可更新代币


### 部署示例
```
npx hardhat run .\scripts\deploy.ts
```
